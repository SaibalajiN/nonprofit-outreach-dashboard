# Nonprofit Outreach Analytics System

## Overview
An end-to-end data analytics system simulating nonprofit outreach operations, including data ingestion, processing, and interactive dashboard reporting.

## Key Features
- Data pipeline with validation, cleaning, and calculated metrics
- Interactive dashboard with KPIs, trends, and regional insights
- Tracks key metrics such as people served, donations, and engagement
- Designed for non-technical users with intuitive navigation and filters

## Tech Stack
Python, HTML, JavaScript (Chart.js)

## Live Demo
https://saibalajin.github.io/nonprofit-outreach-dashboard/

## Use Case
Built to simulate how nonprofit organizations can monitor outreach performance and make data-driven decisions using a scalable and low-maintenance analytics system.
